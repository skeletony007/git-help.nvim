### git-help.nvim

All the lua functions wrapping Git that I don't want to write twice.

### Instalation

Using [lazy.nvim]

```lua
return {
    "skeletony007/git-help.nvim",

    dependencies = {
        "nvim-lua/plenary.nvim",
    },

    config = true,
}
```

[lazy.nvim]: https://github.com/folke/lazy.nvim
